MySQL-用户&授权&修改端口号

MySQL尝试安装，结果玩坏了，安装一直报错，下面这些操作是删除安装失败后残留的文件的，然后老老实实的用sudo apt-get install mysql-server安装的。可能有一些命令是重复的，不过我也不知道是什么意思，不敢随便动。

mysql为root设置密码

这里有一点疑问：在安装数据库的机器上使用root登陆好像不输入密码就可以，输入错的密码也可以，不知道什么情况。。

host：指定该用户在哪个主机上可以登陆，如果是本地用户可用localhost，如果想让该用户可以从任意远程主机登陆，可以使用通配符%

password：该用户的登陆密码，密码可以为空，如果为空则该用户可以不需要密码登陆服务器

privileges：用户的操作权限，如SELECT，INSERT，UPDATE等，如果要授予所的权限则使用ALL

databasename：数据库名

tablename：表名，如果要授予该用户对所有数据库和表的相应操作权限则可用*表示，如*.*

mysql创建schema

这里有一个问题，schema 和 database 有啥区别。。

删除用户

Delete FROM user Where User='test' and Host='localhost';flush privileges;

删除账户及权限

drop user 用户名@'%';

列出所有用户

SELECT DISTINCT CONCAT('User: ''',user,'''@''',host,''';') AS query FROM mysql.user;

修改端口号

1. 查看端口号

show global variables like 'port';

2. 修改配置文件

vim /etc/mysql/mysql.conf.d/mysqld.cnf

搜”3306”改了就行

3. 重启服务

service mysql restar
